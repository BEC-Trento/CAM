Module Hierarchy

  • DataPanel: Plot measurement results.
  • FitResultTableGrid: Store and display fit results in table.
  • IMPACT: High level interface to Matrix Vision cams (mvBlueFOX).
  • ImagePanel: Display images.
  • ImageTree: Provides classes for representing and displaying list of saved image files as tree.
  • LM: This module contains pure Python implementations of the Levenberg-Marquardt algorithm for data fitting.
  • SIS: High level interface for Theta-System cameras.
  • SISlib: Low level wrapper to SisApi.
  • acquire: Acquire images, display and save them.
  • avgdata: Calculate averages and errors.
  • cam: Main file for camera program.
  • ding: Play a (random) sound.
  • fitting: Module for fitting routines and representation of fit results.
  • gridtypes: Custom Editors and Renderers for grid entries.
  • imagingpars: Contains imaging parameters like effective pixelsize, absorption coefficient, ...
  • observer: Provides mixin classes for the observer design pattern.
  • readsis: Load image files which have been saved by WinSIS.
  • roi: Region Of Interest.
  • toolbar: Improved implementation of NavigationToolbar for matplotlib.
